DGZ Motorshop - Minimal Web App (PHP + MySQL)
Place this folder inside XAMPP's htdocs or your PHP webroot.

Steps:
1. Import 'init.sql' into your MySQL (phpMyAdmin) to create the database and sample data.
   - Database name: dgz_db
2. Edit 'config.php' to set DB username/password if needed.
3. Start Apache & MySQL (XAMPP). Access:
   - Customer shop: http://localhost/dgz_motorshop_system/index.php
   - Admin: http://localhost/dgz_motorshop_system/admin/login.php
   - Default admin login: admin@example.com / password123

Notes:
- This is a minimal, educational starter system. Secure and harden before production.
